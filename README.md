# Coastal Futures Summit 2026 - Event Registration Portal

A modern, responsive event registration portal for the Coastal Futures Summit 2026, a one-day gathering of makers, designers, and technologists shaping the next generation of sustainable cities.

## Features

### Event Information
- **Event**: Coastal Futures Summit 2026
- **Focus**: Sustainable cities and climate technology
- **Format**: Both in-person and virtual attendance options
- **Target Audience**: Makers, designers, and technologists

### Registration Features
- **Complete Registration Form** with validation
- **Multiple Tracks**: Design & Urban Futures, Climate Tech, Policy & Partnerships, Maker Labs
- **Attendance Modes**: In-person and virtual options
- **File Upload**: Badge photo upload functionality
- **Special Requirements**: Accessibility and dietary needs accommodation

### Technical Features
- **Responsive Design**: Mobile-first approach with CSS Grid and Flexbox
- **Modern CSS**: Custom properties (CSS variables) for theming
- **Smooth Animations**: CSS keyframe animations for enhanced UX
- **Form Validation**: HTML5 form validation with required fields
- **Media Integration**: Video and audio content support
- **Accessibility**: Semantic HTML structure

## Design System

### Color Palette
- **Ink**: #1f2a2e (Primary text)
- **Sand**: #f7f1e6 (Background accent)
- **Coral**: #e0685a (Call-to-action)
- **Ocean**: #1f6f7a (Secondary accent)
- **Leaf**: #2f7c5b (Success states)
- **Card**: #fff8ef (Card backgrounds)

### Typography
- **Headings**: Playfair Display (Serif)
- **Body Text**: Space Grotesk (Sans-serif)
- **System Fallbacks**: Segoe UI, Times New Roman

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/siva-balanr05/AI-WE-EX1.git
   cd AI-WE-EX1
   ```

2. **Open in browser**
   - Simply open `event.html` in any modern web browser
   - No build process or dependencies required

3. **Local Development**
   - Use any local server (Live Server extension in VS Code recommended)
   - Or open directly in browser for basic functionality

## Browser Compatibility

- **Modern Browsers**: Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
- **Features Used**: CSS Grid, Custom Properties, ES6+ JavaScript
- **Responsive**: Works on desktop, tablet, and mobile devices

## Form Functionality

The registration form includes:
- **Required fields**: Name, email, event date, attendance mode, track selection
- **Optional fields**: Special requirements, badge photo upload
- **Validation**: HTML5 form validation with user feedback
- **File upload**: Image files (JPG/PNG) for badge photos
